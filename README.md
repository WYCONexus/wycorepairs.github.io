# WYCO Repairs Static Site

This is a GitHub Pages-ready static site starter for WYCO Repairs.

## Structure
- `index.html` — homepage
- `services.html` — services and pricing
- `about.html` — business background page
- `contact.html` — includes Jotform embed slot
- `privacy.html`, `terms.html`, `disclaimer.html`, `accessibility.html`, `cookie-policy.html` — legal/footer pages
- `css/style.css` — shared styling
- `js/main.js` — small nav helper

## Jotform setup
Open `contact.html` and replace the placeholder block in the `jotform-slot` area with your actual Jotform embed code.

Example:
```html
<script src="https://form.jotform.com/jsform/XXXXXXXXXX"></script>
```

## GitHub Pages quick start
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In repository settings, enable GitHub Pages from the main branch.
4. Point your domain to GitHub Pages if you want to use `wycorepairs.com`.

## Notes
This is a styled starter build based on your WYCO Hub direction and the current WYCO Repairs site structure. Review all legal language before publishing.
